Prilikom kucanja CIMXML datoteke moguce je izostaviti sledeca polja koja ce biti postavljenja na defaultnu vrednost:
	- <cim:EnergyStorage.state> 		=> default Idle
	- <cim:EnergyStorage.stateOfCharge> => default 1
	- <cim:DER.setPoint> 				=> default 0
	- <cim:DER.activePower>				=> default 0
	- <cim:Generator.deltaPower>        => default 0
